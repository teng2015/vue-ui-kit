TimeFormatter is a filter that uses MomentJS to format time.

It accepts a timestamp and converts the timestamp to human readable time. For example:

Timestamp: {{ time }}

Formatted Time 1 (Default): {{ time | timeFormatter }}

Formatted Time 2 ('YYYY/MM/DD'): {{ time | timeFormatter 'YYYY/MM/DD' }}

Formatted Time 3 ('YYYY年MM月DD日 HH时mm分ss秒'): {{ time | timeFormatter 'YYYY年MM月DD日 HH时mm分ss秒' }}
