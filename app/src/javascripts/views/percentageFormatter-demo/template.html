PercentageFormatter is a filter that converts a number to percentage.

Input a number, eg: 0.76.

Rounded to 2 decimals, keep 0s at the end (Default): {{ number | percentageFormatter }}

Rounded to 4 decimals, keep 0s at the end: {{ number | percentageFormatter '4' }}

Rounded to 2 decimals, omit 0s at the end (Pass a second parameter 'true'): {{ number | percentageFormatter '2' 'true' }}
